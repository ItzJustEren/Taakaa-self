🤖 TaaKaa-Self Bot

یه ربات ساده و کاربردی برای تلگرام که بهت کمک میکنه پیام‌های تکراری رو به صورت خودکار در گروه‌ها یا کانال‌ها ارسال کنی!

---

## 📖 توضیحات کامل

### این ربات چیکار میکنه؟

این ربات با استفاده از **اکانت شخصی** شما وارد تلگرام میشه و هر چند دقیقه که شما تنظیم کنید، یه پیام مشخص رو به یه گروه یا کانال خاص میفرسته.

### چه کاربردی داره؟

- **یادآوری خودکار**: مثلاً هر روز یه پیام یادآوری بفرسته
- **ربات بازی**: توی بازی‌ها هر چند دقیقه یه کلمه بفرسته تا امتیاز بگیره
- **اطلاع‌رسانی**: هر چند وقت یه پیام اطلاع‌رسانی بفرسته
- **سرگرمی**: مثلاً هر ۵ دقیقه بگه "میو" 😸

### تفاوت با ربات‌های معمولی

- **با اکانت شخصی** کار میکنه، نه ربات
- **نیاز به API ID و API HASH نداره** (با QR Code وارد میشه)
- **مثل یه کاربر عادی** پیام میفرسته
- **توی گروه‌ها و کانال‌ها** میتونه پیام بفرسته

---

## ⚠️ نکات مهم قبل از شروع

1. این ربات با **اکانت شخصی** شما کار میکنه
2. برای استفاده در گروه‌ها، حتماً **اجازه ارسال پیام** داشته باش
3. از ارسال **اسپم** خودداری کن (ممکنه اکانتت محدود بشه)
4. فایل `session.session` رو **نگهداری** کن تا دیگه QR نخواد
5. این ربات **قانونی** هست و برخلاف Self-Bot های قبلی نیست

## 🚀 آموزش راه‌اندازی گام‌به‌گام

### مرحله ۱: نصب پایتون

اگه پایتون روی سیستمت نصب نیست:

1. برو به سایت [python.org](https://python.org)
2. آخرین نسخه رو دانلود کن
3. هنگام نصب، حتماً گزینه **"Add Python to PATH"** رو تیک بزن
4. نصب رو کامل کن

برای چک کردن نصب، توی CMD یا ترمینال بنویس:
```bash
python --version
```

اگه نسخه رو نشون داد، یعنی نصب شده ✅

---

مرحله ۲: دانلود پروژه

روش اول: با گیت (پیشنهادی)

```bash
git clone https://github.com/ItzJustEren/TaaKaa-Self
cd TaaKaa-Self
```

روش دوم: دانلود دستی

1. برو به github.com/ItzJustEren/TaaKaa-Self
2. بزن Code → Download ZIP
3. فایل رو از حالت فشرده خارج کن
4. توی CMD برو به پوشه‌ی پروژه:

```bash
cd مسیر-پوشه-پروژه
```

---

مرحله ۳: نصب پیش‌نیازها

توی CMD یا ترمینال، توی پوشه‌ی پروژه این دستور رو بزن:

```bash
pip install telethon
```

اگه خطا داد، این رو امتحان کن:

```bash
pip3 install telethon
```

---

مرحله ۴: اجرای ربات

```bash
python bot.py
```

اگه خطا داد:

```bash
python3 bot.py
```

---

مرحله ۵: ورود با QR Code

بعد از اجرا، یه QR Code تو ترمینال ظاهر میشه:

1. تلگرام گوشی رو باز کن
2. برو به Settings (تنظیمات) → Devices (دستگاه‌ها) → Scan QR Code
3. QR Code رو اسکن کن
4. تبریک! وارد شدی ✅
5. یه فایل session.session ساخته میشه که دفعه بعد دیگه QR نمیخواد

---

مرحله ۶: تنظیم ربات

حالا برو به Saved Messages (پیام‌های ذخیره شده) خودت توی تلگرام:

1. بنویس /start
2. آیدی یا لینک گپ رو بفرست:
   · مثلاً: @mygroup
   · یا: https://t.me/mygroup
   · یا: عدد آیدی گروه
3. زمان ارسال رو به دقیقه وارد کن:
   · مثلاً: 5 (یعنی هر ۵ دقیقه)
4. متن پیام رو وارد کن:
   · مثلاً: میو
5. ربات شروع به کار میکنه! 🚀

---

مرحله ۷: توقف ربات

برای متوقف کردن ارسال پیام‌ها، توی Saved Messages بنویس:

```
/stop
```

برای خروج کامل از ربات، توی CMD بزن Ctrl+C

---

📱 نحوه پیدا کردن آیدی گپ

روش ۱: با یوزرنیم (ساده‌ترین)

اگه گروه یا کانال یوزرنیم داره، فقط با @username وارد کن:

· مثال: @mygroup

روش ۲: با لینک

لینک گروه یا کانال رو کپی کن:

· مثال: https://t.me/mygroup

روش ۳: با ID عددی

1. توی تلگرام برو به گروه یا کانال
2. یه پیام به ربات @userinfobot بفرست
3. ربات بهت ID عددی رو میده
4. همون عدد رو وارد کن

❓ سوالات متداول

چطور بفهمم ربات داره کار میکنه؟

توی CMD پیام‌های "✅" رو می‌بینی که میگه پیام فرستاده شده.

چرا پیام نمیفرسته؟

1. مطمئن شو توی گروه اجازه ارسال داری
2. آیدی گپ رو درست وارد کرده باشی
3. ربات رو با /stop متوقف نکرده باشی

چطور دیگه QR نخواد؟

فایل session.session که بعد از اولین لاگین ساخته میشه رو نگه دار. تا وقتی هست، دیگه QR نمیخواد.

آیا این ربات قانونیه؟

بله! این روش با QR Code ورود، کاملاً قانونی و مطابق با قوانین تلگرام هست.

ممکنه اکانتم محدود بشه؟

اگه اسپم نکنی و فاصله زمانی مناسب بذاری، خطری نداره.
